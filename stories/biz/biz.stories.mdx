# 表格功能拓展

ali-react-table 提供了业务开发经常需要用到的一些功能，例如：

- 自定义列
- 多列排序
- Excel 文件导出
- 树状模式
- 列高亮等

部分功能（多列排序、树状模式等）通过拓展表格功能（拓展的方式详见 transform）来进行提供，而其他功能由导出普通的函数或 React 组件进行提供。

表格功能拓展相关的导出符号较多且不同版本之间变化较大，注意安装的版本。
